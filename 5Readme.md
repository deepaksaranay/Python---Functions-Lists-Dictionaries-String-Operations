# Assignment 5

This assignment has two tasks.

Task 1:
A dictionary is created with student names and marks.
The user enters a name.
If the name is found, marks are displayed.
If not, it shows "Student not found".

Task 2:
A list from 1 to 10 is created.
First five elements are taken.
These elements are reversed.
All lists are printed.

How to run:
1. Open terminal or command prompt
2. Go to the file location
3. Run:
python assignment5.py
